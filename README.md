# like ls command 📁
CLIの練習で作ったLSコマンドです。

## Usage
```md
Usage:
  gols [options][file...]

unix ls command for golang

Application Options:
  /v, /version   Show version
  /r, /recurse   recurse
  /p, /path:     path (default: ./)

Help Options:
  /?             Show this help message
  /h, /help      Show this help message
```

## Install
`go install github.com/koutarn/gols`
